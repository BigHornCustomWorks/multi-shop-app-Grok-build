# Custom Shop Management

Multi-tenant web app for auto body / custom shops:

- **Master Control** (platform owner only) — create shops, logos (JPG upload), locations, statuses, AI upgrade flag  
- **Shop floor** — jobs, vehicle location, parts + parts location, notes, photos  
- Mobile-friendly browser app (phone / tablet / desktop)

## Quick start

1. Install [Node.js LTS](https://nodejs.org)  
2. Follow **SETUP.md** (new Firebase project + rules)  
3. Run:

```powershell
cd "I:\The Stussis\Documents\Big Horn Custom Works\Shop Management App"
npm install
npm run dev
```
